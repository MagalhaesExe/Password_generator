# Password Generator

Um gerador de senhas seguro, moderno e responsivo, desenvolvido com HTML, CSS e JavaScript puro. O projeto conta com indicadores de força da senha, personalização de caracteres e suporte a modo escuro.

## Funcionalidades

* **Geração Personalizada:** Escolha o comprimento da senha e inclua letras maiúsculas, números e símbolos.
* **Indicador de Segurança:** Uma barra visual que indica se a senha gerada é fraca, média ou forte.
* **Cópia Rápida:** Botões para copiar a senha gerada diretamente para a área de transferência.
* **Modo Escuro (Dark Mode):** Interface adaptável com preferência salva no navegador (LocalStorage).
* **Design Responsivo:** Otimizado para dispositivos móveis e desktop.
* **Ajuste Dinâmico:** O tamanho da fonte da senha se ajusta automaticamente para caber na tela.

---

## Tecnologias Utilizadas

* **HTML5**: Estrutura semântica.
* **CSS3**: Estilização com Variáveis CSS (Custom Properties), Flexbox e Media Queries.
* **JavaScript (ES6+)**: Lógica de geração de senhas, manipulação do DOM e LocalStorage.
* **SVG Icons**: Ícones leves e escaláveis para uma melhor UI.

---

## Estrutura do Projeto

O código foi organizado seguindo as melhores práticas de separação de responsabilidades:

- `index.html`: Estrutura da interface.
- `style.css`: Estilos globais e temas.
- `reset.css`: Reset para garantir consistência entre navegadores.
- `script.js`: Toda a lógica e interatividade do sistema.
- `LICENSE`: Arquivo da licença MIT

## Como executar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/MagalhaesExe/Password_generator.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd password_generator
   ```
3. Abra o arquivo `index.html` no seu navegador de preferência.
